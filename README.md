# DemoVue3

To follow demo step by step: You can run each script in vue-demo-scripts individually to follow progression.

```bash
./vue-demo-scripts/0_vuejs_setup.sh
./vue-demo-scripts/1_...
./vue-demo-scripts/2_...
```

To get the complete version:

```bash
./build_demo.sh
```
